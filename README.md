# Pizza Sales Analysis Using SQL

## Project Overview
Performed sales analysis on pizza store data using SQL to identify revenue trends, top-selling pizzas, customer ordering patterns, and business insights.

---

## Tech Stack
- SQL
- MySQL
- Microsoft Excel

---

## Features
- Revenue Analysis
- Top-Selling Pizza Identification
- Order Trend Analysis
- Category-Wise Sales Analysis
- KPI Reporting
- Data Aggregation and Filtering

---

## SQL Concepts Used
- JOINS
- GROUP BY
- ORDER BY
- Aggregate Functions
- Subqueries

---

## Insights Generated
- Identified highest revenue-generating pizza categories
- Analyzed peak order timings
- Calculated total revenue and average order value
- Determined top 5 best-selling pizzas
